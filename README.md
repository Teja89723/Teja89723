A full-stack Job Portal developed using **Spring Boot**, **MySQL**, **Spring Security**, and **Thymeleaf**, enabling recruiters to post jobs and job seekers to search and apply seamlessly.

- 👨‍💼 Recruiter Dashboard
- 👨‍🎓 Job Seeker Dashboard
- 🔐 Spring Security Authentication

- 📊 Profile Management
